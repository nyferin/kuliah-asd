# TREE
  Kumpulan node yang saling terhubung dalam satu kesatuan yang membentuk layaknya struktur sebuah pohon.
  
  Jenis transverse:
  a. PreOrder
     - Root -> Left -> Right
  b. InOrder
     - Left -> Root -> Right
  c. PostOrder
     - Left -> Right -> Root
